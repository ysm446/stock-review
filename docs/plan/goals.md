# Goals

## プロジェクトの目的

個人の株式投資を「記録・可視化・振り返り」するためのデスクトップアプリ（Electron 製）。
保有銘柄の管理と、個別銘柄のレビュー（ファンダメンタル分析メモ）を一つのツールで完結させることを目指す。

## 完成形のイメージ

- **Portfolio ビュー**: 保有銘柄の入力・評価額の自動計算・保有割合チャート・損益の可視化・資産推移トレンド。
- **Review ビュー**: 個別銘柄の企業指標（PER/PBR/ROE など）とニュースを取得し、ウォッチリスト形式でレビューを蓄積。
- **企業指標ビュー**: 保有 / ウォッチリスト銘柄の指標を一覧で比較。
- **LLM チャット**: ポートフォリオ全体・個別銘柄について対話的に相談できる。
- 株価・FX・セクター・配当は `yfinance` から取得し、外貨建ては自動で円換算。

## 重視する価値

- **正確さ**: 評価額・損益・円換算・配当の計算が常に正しいこと。重複保有や複数通貨を取りこぼさない。
- **見やすさ**: チャート・配色・テーブルが直感的で、状態を一目で把握できること。
- **ローカル完結**: データは `data/` 配下の SQLite / JSON にローカル保存し、外部送信に依存しない。
- **手入力の手間を減らす**: 価格・指標・セクターなどは可能な限り自動取得する。
